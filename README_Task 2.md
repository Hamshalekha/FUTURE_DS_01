# Customer Retention & Churn Analysis

## Project Overview
This project analyzes customer churn patterns using a telecom customer dataset. The goal is to identify retention drivers, churn trends, and provide business recommendations.

## Tools Used
- Python
- Pandas
- NumPy
- Tableau Public

## Key Metrics
- Total Customers: 7032
- Churned Customers: 1869
- Churn Rate: 26.58%
- Retention Rate: 73.42%

## Dashboard Insights
- Month-to-month contracts have the highest churn.
- Fiber Optic customers show the largest churn volume.
- Electronic Check users exhibit the highest churn.
- Senior citizens represent a higher-risk customer segment.

## Recommendations
- Promote long-term contracts.
- Improve Fiber Optic customer experience.
- Encourage automatic payment methods.
- Create targeted retention campaigns.

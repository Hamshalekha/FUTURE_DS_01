Superstore Sales Performance Dashboard

Project Overview

This project was completed as part of the Future Interns Data Science & Analytics Internship (Task 1).

The objective of this project is to analyze sales performance using the Superstore dataset and build an interactive Tableau dashboard.

Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Tableau Public

Dataset Information

* Total Rows: 9,994
* Total Columns: 21

Key columns used:

* Order Date
* Sales
* Profit
* Region
* Category
* Product Name

Key Performance Indicators

* Total Sales: $2,297,200.86
* Total Profit: $286,397.02
* Total Orders: 5,009
* Total Customers: 793
* Average Order Value: $458.61

  
Key Insights

* West region generated the highest sales.
* Technology category generated the highest revenue.
* Canon imageCLASS 2200 Advanced Copier was the top-selling product.
* Sales increased significantly during 2016 and 2017.
